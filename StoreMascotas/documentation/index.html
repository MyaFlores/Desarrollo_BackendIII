🐾 Home Design - Dog Adoption API

API RESTful para gestionar perritos, realizar búsquedas, aplicar filtros dinámicos y consultar disponibilidad para adopción.

REST API JSON responses Filtros avanzados

Endpoints disponibles

GET /api/dogs/

Descripción

Obtiene todos los perritos registrados. Soporta filtros y ordenamiento.

Parámetros (query)

  • min_age - Edad mínima en meses
  • max_age - Edad máxima en meses
  • vaccinated - true/false
  • size - small/medium/large
  • adopted - true/false
  • ordering - age, -age, weight, -weight

Ejemplo request

GET /api/dogs/?min_age=1&max_age=12&vaccinated=true&ordering=-weight

Ejemplo response

{
  "message": "Dogs retrieved successfully",
  "total": 2,
  "data": [
    {"id": 1, "name": "Walle", "breed": "Pug", "age": 8, ...}
  ]
}
200 OK 400 Bad Request
GET /api/dogs/{id}/

Descripción

Obtiene un perrito específico por su ID.

Parámetros (path)

  • id - ID del perrito (entero)

Ejemplo request

GET /api/dogs/5/

Ejemplo response (éxito)

{
  "message": "Dog found",
  "total": 1,
  "data": {"id": 5, "name": "Luna", ...}
}

Error (404)

{
  "message": "Dog with id 99 not found",
  "error": "Resource does not exist"
}
200 OK 404 Not Found
